# Advanced Demo Application

![](https://slabstatic.com/prod/uploads/4ojndffx/posts/images/ubiT9LXqonw9z-b4_KLObNI5.png)

This is an advanced demo application that illustrates the embedding of the Tangl viewer and its interface library, as well as containing two example extensions to the viewer.

The project also uses a complete setup that you can use as a basis for developing your own applications

## Overview

See demo application overview on Tangl help center:

[**https://help.tangl.cloud/api/tangl-viewer/tangl-viewer-howto/tangl-demo-advanced-overview.html**](https://help.tangl.cloud/api/tangl-viewer/tangl-viewer-howto/tangl-demo-advanced-overview.html)
